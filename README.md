## 项目简介

本项目是《AI 安全：技术、攻防与实战》一书的完整开源文稿，面向希望系统了解 **AI 安全（AI Security）** 与 **AI 对齐（AI Safety）** 的读者。

全书以**原理讲解 + 关键代码片段 + 实战案例 + 思考拓展**为主线，覆盖从对抗样本、数据投毒、模型后门，到深度伪造、AI Agent 安全等 2020–2026 年主流攻防技术。读者无需深厚的安全背景，只要了解基础的 Python 编程与机器学习概念即可顺畅阅读。

> **为什么开源？** AI 安全是一个仍在快速演化的领域，我们希望通过开源让更多开发者、研究者和学生能够低成本地进入这个领域，共同推动 AI 安全生态的建设。
